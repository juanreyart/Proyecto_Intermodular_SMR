# Montaje y Mantenimiento - Galería de Arte Digital

Este repositorio documenta la selección de hardware, el proceso de ensamblaje de las estaciones de trabajo y el plan de mantenimiento preventivo y correctivo diseñado para la **Galería de Arte Digital**. Se detallan los protocolos para asegurar que el hardware de alto rendimiento se mantenga operativo, estable y protegido ante cualquier eventualidad.

**Juan Antonio Rey Vicente** | Estudiante de 1º SMR | Vilagarcía de Arousa (Galicia)

---

## Índice del Proyecto
1.  **Selección de Componentes:** Análisis del hardware necesario (CPU, RAM, GPU, Almacenamiento) para la visualización 4K.
2.  **Proceso de Montaje:** Guía paso a paso del ensamblaje del chasis, placa base, refrigeración y periféricos.
3.  **Mantenimiento Preventivo:** Protocolos de limpieza interna, control de ventilación y temperaturas.
4.  **Gestión de Incidencias:** Procedimientos para la actualización de componentes y sustitución de piezas defectuosas.
5.  **Copias de Seguridad:** Plan de contingencia mediante el uso de Cobian Backup y almacenamiento externo/nube.
6.  **Inventario de Activos:** Lista completa de los equipos informáticos que conforman la infraestructura.

---

## Arquitectura y Montaje
La infraestructura se basa en equipos de alto rendimiento (i7-14700KF, 32GB RAM DDR5, GPU 16GB) configurados específicamente para renderizar y gestionar obras de arte en 4K. El proceso de montaje ha sido documentado cuidadosamente para garantizar la correcta refrigeración, la gestión de cables y la durabilidad a largo plazo de los componentes críticos.

## Protocolos de Mantenimiento
* **Limpieza y Ventilación:** Revisiones periódicas para evitar la acumulación de polvo en ventiladores y disipadores, garantizando un flujo de aire óptimo.
* **Monitoreo:** Control constante de las temperaturas de trabajo para evitar el *thermal throttling* durante exposiciones prolongadas.
* **Resiliencia:** Plan de respaldo semanal mediante copias de seguridad automatizadas, asegurando la integridad de las obras digitales ante cualquier fallo físico del equipo.
* **Seguridad Eléctrica:** Protección integral de los equipos mediante sistemas SAI (UPS), evitando daños por picos de tensión o cortes eléctricos.

## Resultados y Validación
* **Confiabilidad:** El mantenimiento proactivo reduce drásticamente el riesgo de fallos inesperados en el hardware durante las exposiciones.
* **Gestión de Activos:** El inventario actualizado permite una administración rápida y eficaz de los recursos técnicos en caso de reparaciones o ampliaciones.
* **Optimización:** La correcta configuración y ventilación aseguran que los componentes seleccionados operen a su máximo rendimiento sin degradación prematura.
